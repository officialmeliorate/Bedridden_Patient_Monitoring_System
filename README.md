Smart IoT Based Sign to Text Conversion And Medical Assistance System for bedridden patients, mainly works for allowing doctors or relatives of patient to check the status of patient health remotely and thereby reducing the effort of caretakers.
The system monitors the live health conditions such as heart rate, dissolved oxygen level in blood etc and provides the live result in mobile phones through Blynk cloud.
Node MCU ESP 8266 which is interfaced with Pulse Oximeter MAX30100 for detecting the live B.P level and Oxygen carrying capacity of blood and also contains three IR sensor module for detecting the wave motion corresponding to three messages including, “I Need Food”, ”I Need Water” and “Emergency, Please Come!!”. 
The system works with IoT and Cloud Server interface to send the data remotely to the cloud . The system also introduced a function through which
caretakercan be contacted by gestures.
Identifying the gesture of patient, our system will identify these instructions and display the message as notification on the Blynk application and also send email regarding the message on the email provided.
